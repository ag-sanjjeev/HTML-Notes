## &#9873; Preformatted Text Tag
This `<pre>` tag is used to display the content or text as exactly as written in the HTML code.

*This will preserve the content or text spaces, line breaks and fixed font width.*

### &#9780; Overview:
1. [Syntax](#-syntax),
2. [Attributes and Properties](#-attributes-and-properties),
3. [Event Attributes](#-event-attributes),
4. [Examples](#-examples),
5. [Code Reference](#-code-reference)

### &#10022; Syntax:

Simple `<pre>` tag created as, 
```xml
<pre>{{Content or Text}}</pre>
```

### &#10022; Attributes and Properties:
This can support most of the standard [attributes and properties](../docs/attributes-and-properties.md).

### &#10022; Event Attributes:
This can support most of the standard [events](../docs/events.md).

### &#10022; Examples:
> Example for `<pre>` tag
```xml
<pre>This is sample text</pre>
```
